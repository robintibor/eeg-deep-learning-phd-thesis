Introduction¬∂

Machine learning (ML) can be used for medical applications

  • ML can extract different information from biophysiological signals then humans are able to

  • Applications include assistive technology, diagnosis and scientific discovery

    • Assistive technology requires to quickly extract of information about a persons intention

    • Diagnosis requires to extract information about pathologies and combine it with human knowledge

      • Early-warning requires to quickly extract information from large amounts of continously arriving signals

    • Scientific discovery requires understanding what novel information the ML model has extracted

Brain signals are interesting for medical ML

  • Can allow decoding of intention ‚Äúat source‚Äù

  • Allow a lot of diagnoses

  • Also brain very complicated, and new things may be discovered

Electroencephalography (EEG) has useful properties for ML analysis

  • Cheap to acquire, widely available

  • No side effects

  • Hard to interpret for humans

  • Some applications limited by accuracies achieved so far

    • BCI, diagnosis, scientific discovery

Deep learning is an interesting approach for EEG

  • Can exploit large data well

  • Proven to work well on diversity of natural signals

  • Flexible models/little inductive bias

Deep learning is a challenging approach for EEG

  • Unknown if it is really better than feature-based approaches

  • Unknown how much information can really be extracted from EEG

  • Limited understanding of EEG signal also limits understanding of DL EEG models

  • Non-brain-signals should not be used when testing assistive technology with healthy population

Findings

Prior Work

Prior to 2017, research did not clearly show how competitive deep learning is compared with well-optimized feature baselines

Task-Related Decoding

Deep learning can be at least as good as strong feature-based baselines for task-related decoding; deep networks also learn to extract known hand-engineered features

Decoding Pathology

Deep networks designed for task-related decoding can also decode pathology well

Further Deep Learning Improvements

Deep learning improvements from other domains partially carry over to EEG decoding

Back to Features - a Comparison

Well-optimized feature baselines remain competitive with deep learning

Future Work

Newer DL architectures such as transformers or invertible networks may allow better performance and better interpretability
